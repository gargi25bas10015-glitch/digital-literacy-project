# digital-literacy-project
Digital Literacy Project showcasing skills in online presence, professional communication, cybersecurity awareness, and collaborative tools.
Digital Literacy Project



##  Project Overview
This repository contains my Digital Literacy Project submitted as part of the CSE0001 course at VIT Bhopal University. The objective of this project is to develop essential digital skills required for academic and professional growth.

The project focuses on:
- Understanding digital literacy concepts  
- Building a professional online presence  
- Using coding and collaboration platforms  
- Practicing proper email etiquette  
- Learning about cybercrime and online safety  


##  Tasks Completed

### Task 1: Digital Literacy Infographic
Created a one-page infographic using Canva to explain digital literacy, safe internet practices, and professional online presence.



###  Task 2: Digital Portfolio
Developed professional profiles on:
- GitHub (for projects and coding work)  
- LinkedIn (for professional networking)  
- HackerRank (for coding practice)  



###  Task 3: Coding & Collaboration Platforms
- Completed a beginner-level coding problem on HackerRank  
- Created a Google Form titled **Digital Literacy Awareness Quiz**  



###  Task 4: Email Etiquette
- Drafted two professional emails  
- Created a social media **Do’s and Don’ts checklist**  



###  Task 5: Cybercrime Awareness
- Wrote a case study on **UPI Fraud**  
- Created a cyber safety prevention checklist
